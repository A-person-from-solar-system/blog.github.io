创建一个springboot项目

一、创建一个java项目

二、实现一个基础的springboot程序

基础的springboot程序由三层构成，包括：

  • 表现层(controller类)
  • 业务层(service接口和实现类)
  • 数据层(连接数据库负责CRUD操作)

导入lombok

public class BookController {
    @Autowired
    private BookService bookService;

    @GetMapping("/{id}")
    public String getByid(@PathVariable Integer id) {
        return bookService.getBookById(id).toString();
    }
}

package com.coda.service.impl;

import com.coda.item.Book;
import com.coda.service.BookService;
import org.springframework.stereotype.Service;

@Service
public class BookServiceImpl implements BookService {
    @Autowired
    private BookMapper bookMapper;

    @Override
    public Book getBookById(int id) {
        return bookMapper.getBookById(id);
    }
}

import com.coda.item.Book;
import org.apache.ibatis.annotations.Mapper;
import org.apache.ibatis.annotations.Select;

@Mapper
public interface BookMapper {
    @Select("select * from tb1_book where id=#{id}")
    public Book getBookById(int id);
}

6. 数据库

1）IDEA中设置连接数据库的信息

resource目录下添加application.yml文件：

yml文件的配置内容：

  • com.mysql.cj.jdbc.Driver为数据库驱动

  • jdbc:mysql://localhost:3306/ssm_db

    • ssm_db：为自己创建的数据库名

    • 3306：为数据库的端口

    • root：数据库的名称

    • password：数据库的密码
